Point Class

Defined in: lib/Point.js:11
Module: kami

This is a convenience wrapper around gl-matrix "vec2" type. It is backed by a Float32Array, and the x and y properties are included for convenience.

For performance, or when performing gl-matrix operations, you should access the items array directly. For convenience and clarity, the x and y properties are encouraged.
